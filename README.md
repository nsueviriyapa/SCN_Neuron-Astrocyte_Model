# SCN_Neuron-Astrocyte_Model

This repository contains codes written for MATLAB (R2018a) to simulate the SCN neuron-astrocyte communication.

Reference   

Sueviriyapan, N., Tso, C. F., Herzog, E. D., & Henson, M. A. (2020). Astrocytic Modulation of Neuronal Activity in the Suprachiasmatic Nucleus: Insights from Mathematical Modeling. Journal of Biological Rhythms. https://doi.org/10.1177/0748730420913672
